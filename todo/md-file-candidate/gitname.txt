How to save username and password in git [GitExtension]?


  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"

[dyiwu@pb6440b ~]$ cat ~/.gitconfig
[user]
        email = dyiwu.liu@gmail.com
        name = dyiwu
[credential]
        helper = store
[dyiwu@pb6440b ~]$ git config --global credential.helper store
